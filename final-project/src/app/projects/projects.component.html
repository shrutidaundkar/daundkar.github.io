Name: Identity Chain

  • The Identity Authentication using Ethereum project is a decentralized application (dApp) that utilizes the Ethereum blockchain to enable secure authentication and verification of user identities.
  • The project is built using the Solidity programming language and the Truffle development framework, and it employs the concept of smart contracts to ensure that all authentication transactions are transparent, tamper-proof, and immutable.
  • The dApp allows users to create an account, which is stored on the blockchain, and then use that account to authenticate themselves on various websites and applications. The authentication process involves the use of public-private key pairs and cryptographic signatures, which ensures that only the authorized user can access their account.

Name: Fun transalation using Angular.Js

  • Fun translations is a mini project I worked on last year to learn Angular.Js. The project is a web application that accepts normal English messages and translates them into fun languages such as Minionese, Yoda, and Pirate. The project's main goal was to learn Angular.Js and explore the capabilities of an open-source API for language translation.
  • I used Angular.Js to develop the front end of the application, implementing a responsive and user-friendly interface that allows users to enter their messages and select the fun language they want to translate to. An open-source API powered the application's backend for language translation, which I integrated into the application using Angular.Js.
